# Django-Sentiment-Analysis-Application

## Description
This application calculates the sentiment of given text and yelp reviews on a 1-5 scale (very bad - very good).

|Score|Sentiment|
|-|-|
|1| very bad |  
| 2 | bad|
| 3 | meh| 
| 4 | good|
| 5 | very good|
